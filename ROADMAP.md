# 🚀 Roadmap de Desenvolvimento - Workout Tracker

Este documento descreve as próximas etapas de desenvolvimento, melhorias planejadas e visão de longo prazo para o projeto.

## 📅 Cronograma Geral

| Fase | Foco | Previsão |
|------|------|----------|
| **Fase 1** | Melhorias Imediatas (UX/UI e Perfil) | 1-2 Semanas |
| **Fase 2** | Funcionalidades Intermediárias (Dados e Metas) | 2-4 Semanas |
| **Fase 3** | Funcionalidades Avançadas (Social e Wearables) | 4-8 Semanas |
| **Fase 4** | Escala e Monetização | 8+ Semanas |

---

## 🚧 Fase 1: Melhorias Imediatas (Q1)
*Foco em Personalização e UX*

- [ ] **Perfil do Usuário**
    - [ ] Edição de dados (Peso, Altura, Idade)
    - [ ] Upload de Foto de Perfil
    - [ ] Gestão de conta (Alterar senha/Deletar)
- [ ] **Interface & UX**
    - [ ] Dark Mode 🌙
    - [ ] Loading Skeletons e Animações
    - [ ] Toast Notifications (Feedback visual)
- [ ] **Performance**
    - [ ] Lazy Loading de rotas
    - [ ] Otimização de queries Firestore

---

## 🛠 Fase 2: Funcionalidades Intermediárias (Q2)
*Foco em Dados e Retenção*

- [ ] **Dashboard & Estatísticas**
    - [ ] Gráficos de progresso (Chart.js/Recharts)
    - [ ] Volume total de carga
- [ ] **Gamificação**
    - [ ] Sistema de Metas e Conquistas
- [ ] **Planos de Treino**
    - [ ] Criação de rotinas (4/8/12 semanas)
- [ ] **Ferramentas de Treino**
    - [ ] Temporizador de descanso
    - [ ] Notas e observações por exercício

---

## 🔮 Fase 3: Funcionalidades Avançadas
*Foco em Comunidade e Integração*

- [ ] **Social:** Sistema de amigos e compartilhamento de treinos.
- [ ] **Wearables:** Integração (Apple Health/Google Fit).
- [ ] **Saúde:** Módulo de Nutrição e Análise de Força (1RM).
- [ ] **Dados:** Backup e Exportação (CSV/PDF).

---

## 💼 Fase 4: Monetização e Expansão
- [ ] Planos Premium (Stripe/PayPal).
- [ ] Painel para Personal Trainers.
- [ ] App Mobile (React Native).
- [ ] IA para recomendações de treino.

---

## 📊 Stack Recomendado para Expansão

* **Gráficos:** Recharts ou Chart.js
* **Notificações:** React-Hot-Toast
* **Formulários:** React Hook Form + Zod
* **Testes:** Vitest + React Testing Library
